# Background

R is a programming language designed for statistical computing and data analysis.

RStudio is an Integrated Development Environment (IDE) that makes writing and running R code much easier.

They are different software:

- **R** performs the calculations.
- **RStudio** provides a user-friendly interface for writing and managing R programs.

RStudio cannot function without R being installed first.

# Laboratory Practical 1 - Installing R and RStudio

## Before you Begin Cehck these

- Internet connection
- Window computer
- Adminstrator access

---

## Downloading R

1. Go to https://cran.r-project.org 

2. Click **Download R for Windows**.

3. Click **base**.

4. Click **Download the latest version of R**.

5. Run the downloaded file.

---

## Installing R

1. Click **Next**.

2. Accept the license.

3. Keep the default installation folder.

4. Click **Install**.

5. Click **Finish**.

---

## Verify Installation

Open R.

If the R console open successfully, the installation is completer.

---

## Common Problems

- R does not open.
- Restart the computer.
- Reinstall R if necessary.

---

## Download RStudio

1. Go to https://posit.co/download/rstudio-desktop/

2. Under **RStudio Desktop**, click **Download**.

3. Select the installer for **Windows**.

4. Wait for the download to complete.

---

## Installing RStudio

1. Open the download RStudio installer.

2. Click **Yes** if Windows asks for permission.

3. Click **Next**.

4. Keept the default installation location.

5. Click **Install**.

6. Wait for the installation to finish.

7. Click **Finish**.

---

## Opening RStudio

1. Open the Start Menu.

2. Search for **RStudio**.

3. Open the application.

4. Wait for RStudio to load completely.

## Verify Installation

The installation is successful if:

- RStudio opens without any errors.
- The console window is visible.
- No message appears saying that R could not be found.

## Notes

- Install **R** before installing **RStudio**.
- RStudio automatically detects an existing R installation.
- Keep the default installation settings unless there is a specific reason to change them.

## Difference Between R and RStudio

| R | RStudio |
|---|---------|
| Programming language | Integrated Development Environment (IDE) |
| Executes code | Provides an interface for writing and managing code |
| Can run without RStudio | Cannot run without R |
