# Laboratory Practical 1 - Installing R and RStudio

---

# Objective

By the end of this practical, I should be able to:

- Understand the difference between R and RStudio
- Install R
- Install RStudio
- Open RStudio successfully
- Prepare my computer for bioinformatics analysis

---

# Background

R is a programming language designed for statistical computing and data analysis.

RStudio is an Integrated Development Environment (IDE) that makes writing and running R code much easier.

They are different software:

- **R** performs the calculations.
- **RStudio** provides a user-friendly interface for writing and managing R programs.

RStudio cannot function without R being installed first.
