# R Programming

Welcome to my R Programming laboratory handbook.

This section documents my journey of learning R programming for bioinformatics and developmental biology.

Unlike traditional workshop notes, every topic in this handbook includes:

- Step-by-step practical instructions
- Why each step is performed
- Expected output
- Common beginner mistakes
- Bioinformatics applications
- Developmental biology examples
- My own practise and observations

# Goal

The goal of this chapter is not simply to learn R syntax.

Instead, I want to understand how R is used to analyse biological data and prepare myself for future research in developmental biology, embryology, infertility and genomics.
