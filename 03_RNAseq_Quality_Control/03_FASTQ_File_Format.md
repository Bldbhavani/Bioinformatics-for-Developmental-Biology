# What is a FASTQ File?

A FASTQ file is the standard file format used to store raw sequencing data generated by Next Generation Sequencing (NGS) platforms.

It contains two important types of information for every sequencing read:

- The nucleotide sequence
- The quality score for every nucleotide

Unlike a FASTA file, FASTQ fiels include quality information that helps determine how reliable each base cell is.

FASTQ is therefore the universal starting point for almost every RNA-seq analysis workflow.
