  ## Why am I learning Linux?
  
  Linux is the most widely used operating system in bioinformatics because most bioinformatics software is designed to run efficiently in a Linux environment.

  Understanding Linux is essential for analysing large biological datasets such as RNA-seq, DNA-seq, ATAC-seq, and single-cell sequencing data.

  ## Goal

  Build a strong Linux foundation for bioinformatics and future research in developmental biology.
